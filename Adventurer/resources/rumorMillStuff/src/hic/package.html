Provides all front-end activities for user interaction, main window, command input, help window, splash screen, and start-up activities. Human Interface (Component) is implemented in this package. Dgncontains the main method and all the global data (such as file directories) and static methods (like output message methods for errors and debugging). (Some static data in the HIC package is made available to the DMC.)

The static main method checks command line arguments and starts the splash window and the main window. The splash window has its own thread to run background tasks like init() and load().

The MainFrame class collects user commands and pushes them into the CommandParser, which waits until the PDC Scheduler polls for them. The Swing thread of the MainFrame windowing system interleaves with the thread of the Scheduler to retrieve and execute the commands created.
