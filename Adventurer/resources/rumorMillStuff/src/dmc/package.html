Provides all SAX Parsing back-end activities for XML and XSD file input, parsing, and validation. Data Management (Component) is implemented in this package. SAXStream reads in the XML data file and validates it against the XSD Schema file. The SAX Parser that is created requires an author-written error handler (MyErrorHandler) to handle any parsing or validating errors that occur; and a content handler (DungeonContentHandler) to move the data into the appropriate objects.

DgnBuild creates the SAX Parser, reads the data files, connect the callback objects, and inits the Inn and other PDC objects.
