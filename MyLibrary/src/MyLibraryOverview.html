myLib Contains the application independent classes, such as the MsgCtrl class for audit and error messages during JUnit tests, a Help engine, reusable GUI widgets, and other helper classes.

NOTE: There is much commonality between Carolla's two flagship programs, Quest Master, the authoring tool, and Adventurer, the Player program. This development library contains general purpose reusable classes, but not ones that are specific to Chronos. Chronos-specific classes will NOT be kept in the library if they are not necessarily resusable for most programs. See the Chronos library.
